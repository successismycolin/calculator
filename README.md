# Simple Calculator

A lightweight calculator web app built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division
- Decimal entry support
- Clear display (`C`) and clear entry (`CE`)
- Prevents invalid operator sequences
- Clickable button interface with instant result display

## Usage

1. Open `index.html` in your browser.
2. Click the number and operator buttons to build an expression.
3. Press `=` to calculate the result.
4. Use `C` to clear the full display or `CE` to remove the last character.

## Project structure

- `index.html` — calculator UI and button layout
- `style.css` — page styling and layout
- `script.js` — calculator logic and input validation

## Development

- Open `index.html` directly in a browser to run the app.
- Edit `style.css` to change the layout and colors.
- Edit `script.js` to update validation rules, button behavior, or calculation logic.

## Notes

- Keyboard input is disabled to keep interactions consistent with the button interface.
- The calculator currently supports single binary operations (for example, `12 + 7`).
- For future improvements, you can add multi-step expressions, keyboard support, or more math functions.
